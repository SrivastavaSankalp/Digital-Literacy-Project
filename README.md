## Project Overview
This repository includes my complete portfolio for the **Student Digital Ambassador Project**. The main goal of this project was to develop responsible digital habits, build a professional online presence, and show practical understanding of areas like cybersecurity, cloud collaboration, and proper online communication.

## Project Summary
This project highlights the key digital skills that a modern computer science student should have. It is divided into five main parts: creating simple visual learning materials, building professional profiles on platforms like Kaggle, ResearchGate and Stack Overflow, completing basic coding tasks in Java, learning how to write proper professional emails, and studying real-world cybersecurity issues like UPI payment fraud to understand how to prevent them.
